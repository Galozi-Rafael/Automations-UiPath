# Automação de Busca e Extração de Invoices

A seguinte automação tem por objetivo navegar para o site ACME na web, realizar login, pesquisar uma invoice específica, extrair seus dados e retornar à tela de busca para novo processamento.

## Objetivos

Navegar para o site ACME;  
Realizar login no sistema;  
Acessar a área de busca de invoices;  
Pesquisar uma invoice específica;  
Extrair os dados da invoice encontrada;  
Retornar à tela de busca para novo processamento.  

## Benefícios da Automação

Evitar trabalho repetitivo de login e navegação;  
Reduzir erros humanos na busca e extração de dados;  
Melhorar o tempo de processamento de invoices;  
Padronizar o fluxo de consulta e extração de informações;  
Possibilitar rastreabilidade e auditoria do processo.  

## Tecnologias Utilizadas

UiPath Studio;  
Automação Web (RPA);  
Seletores e atividades de automação de interface;  
Manipulação de dados.
